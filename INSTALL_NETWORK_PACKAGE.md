Run this command to install the network connectivity package:


Or if that doesn't work:

```bash

```

## What This Package Does

- Detects if device has internet connection
- Checks WiFi, cellular, and other network types
- Provides real-time network status
- Used by SOS feature to decide between FaceTime and phone calls
